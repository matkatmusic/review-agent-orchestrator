## 1. spawnWorktree.sh — Execution Order

```
STEP 1: Parse + validate
  - WORKTREE_NAME="${1// /_}"
  - Validate: [a-zA-Z0-9_-]+ only
  - REPO_ROOT, WORKTREE_PATH="${REPO_ROOT}/../${WORKTREE_NAME}"

STEP 2: Determine state
  - path_exists:   [ -d "${WORKTREE_PATH}" ]
  - branch_exists: git show-ref --verify --quiet "refs/heads/${WORKTREE_NAME}"

STEP 3: Handle state matrix
  CASE A (both exist) — REOPEN:
    - Compare .claude/settings.json and .vscode/tasks.json to templates
    - If modified: prompt "Overwrite? [y/N]"
    - If yes or unchanged: overwrite scaffold
    - → Step 5 (open IDE)

  CASE B (branch exists, no path) — RE-CREATE:
    - git worktree prune
    - git worktree add "${WORKTREE_PATH}" "${WORKTREE_NAME}"
    - → Step 4 (scaffold)

  CASE C (path exists, no branch) — ERROR:
    - Print error + manual cleanup instructions
    - Exit 1

  CASE D (neither) — FRESH CREATE:
    - git worktree add -b "${WORKTREE_NAME}" "${WORKTREE_PATH}" HEAD
    - → Step 4 (scaffold)

STEP 4: Scaffold worktree
  - mkdir -p .claude, .vscode
  - cp v1/templates/settings.json → .claude/settings.json
  - sed "s/__NAME__/${WORKTREE_NAME}/g" v1/templates/tasks.json → .vscode/tasks.json

STEP 5: Open IDE
  - agy --new-window "${WORKTREE_PATH}"
  - IDE auto-runs tasks.json on folder open:
    1. npm install (blocking)
    2. Claude + tmux side-by-side (after npm install, via dependsOn)

STEP 6: Ensure daemon running
  - pgrep -f "worktreeDaemon.sh.*$(basename "${REPO_ROOT}")"
  - If not running: nohup start as fallback (main route is IDE runOnFolderOpen)

STEP 7: Print summary
```

## 2. worktreeDaemon.sh — Main Loop

```
INIT:
  - Parse REPO_ROOT, VERBOSE, POLL_INTERVAL=5
  - Singleton: pgrep for other daemon instances (no PID file)
  - Trap SIGINT/SIGTERM → log + exit

LOOP (every 5s):

  M.1: CHECK MAIN IDE ALIVE
    - pgrep -f "agy.*$(basename "${REPO_ROOT}")"
    - If dead → log "Main IDE closed" → exit 0
    - (Also dies naturally via SIGHUP when IDE terminal closes)

  M.2: DISCOVER WORKTREES
    - git worktree list --porcelain → array of (path, branch)
    - Skip main worktree (first entry)

  M.3: FOR EACH WORKTREE:

    M.3.1: CHECK WORKTREE IDE ALIVE
      - pgrep -f "agy.*${worktree_path}"
      - If dead + tmux session exists → kill tmux session (R7)

    M.3.2: CHECK MERGE STATUS (R8)
      - Call is_worktree_merged() — see Section 4

    M.3.3: IF MERGED → CLEANUP
      a) Remove scaffold files (.claude/settings.json, .vscode/tasks.json)
      b) Check for OTHER dirty files (git status --porcelain, filter scaffold)
      c) If dirty: log WARNING, skip deletion
      d) If clean: kill tmux, git worktree remove --force, git branch -D, git worktree prune

  M.4: sleep 5
```

## 4. is_worktree_merged() — Merge Commit Parent Scan

Scans ALL merge commits for one that has the worktree branch tip as a non-first
parent. This means someone ran `git merge --no-ff worktree_branch`.

No false positives on freshly-created branches (no merge commit exists yet).
No lockfiles. No grace period.

```bash
is_worktree_merged() {
  local branch_name="$1"
  local repo_root="${2:-${REPO_ROOT}}"
  local branch_tip
  branch_tip=$(git -C "${repo_root}" rev-parse "refs/heads/${branch_name}" 2>/dev/null) || return 1

  while IFS= read -r line; do
    read -ra parts <<< "${line}"
    # parts[0]=merge_hash, parts[1]=first_parent, parts[2..n]=merged-in parents
    for (( i=2; i<${#parts[@]}; i++ )); do
      [ "${parts[$i]}" = "${branch_tip}" ] && return 0
    done
  done < <(git -C "${repo_root}" log --all --merges --format='%H %P' 2>/dev/null)

  return 1
}
```

Edge cases handled:
- Octopus merges: checks all non-first parents
- Commits after merge: tip changes, old merge commit no longer matches → correct (not merged)
- Branch deleted: rev-parse fails → returns 1
