/octo:debate — Multi-Agent Invocation Workflow

When invoking the /octo:debate skill, execute agents via Bash with the following steps:

Step: identify the topic being debated, and remember it as <topic>
Step: create a file called <topic>_debate_prompt.txt in the repo root folder /Debates/ and write the debate prompt to it, hereafter known as <file.txt> 


Step: Launch gemini/codex agents in parallel using Bash

  gemini -m <preferred_model> < <file.txt> 

  codex exec -m <preferred_model> -c 'model_reasoning_effort="high"' < <file.txt> 

Step: spawn a claude background agent with the following command:
  Agent(
    subagent_type: "octo:personas:security-auditor",
    description: "claude debate agent",
    prompt: "Read <file.txt> and follow the instructions contained within it.",
    run_in_background: true
  )

Step: 
  Write each agent's debate output into a separate file in the repo root folder /Debates/:
    <topic>_debate_results_gemini.txt
    <topic>_debate_results_codex.txt
    <topic>_debate_results_claude.txt

Step: Summarize
  Read all 3 result files and produce a unified summary of the debate.

Gemini preferred model: gemini-3.1-pro-preview
Codex preferred model: gpt-5.4
