a data structure to represent a message thread: 
<Data Structures>
Message
{
    author: string
    body: string
    timestamp: datetime
    seen: datetime | null
}

Response //a node in a linked list
{
    content: Message
    responding_to: Response | null //(previous message in chain)
    response: Response | null //down arrow (next message in chain)
    replying_to: Response | null //(parent of this thread)
    reply: Response | null //right arrow (reply to this message)
}

Issue
{
    inum: number
    group: string
    blocked_by: Issue[]
    blocks: Issue[]
    body: Response //a linked list root node
}
</Data Structures>

<Issue Creation> 
- create a Message message = {
    .author = agent | user
    .body = <the text> 
    .timestamp = now()
    .seen = null
}
- Create a Response response = {
    .content = message,
    .responding_to = null
    .response = null
    .reply = null
}
- create an Issue issue = {
    .inum = <the number>,
    .group = "",
    .block_by = []
    .blocks = []
    .body = response
}
</Issue Creation> 

<Add Response to Issue>
- find the last Response node in the issue: 
lastResponseNode = issue.body
while( lastResponseNode != null )
    if( lastResponseNode.response == null )
        break;
    lastResponseNode = lastResponseNode.response 
- create a Message message = {
    .author = agent | user
    .body = <the text> 
    .timestamp = now()
    .seen = null
}
- Create a Response new_response = {
    .content = message,
    .responding_to = the end of the Response linked-list
    .response = null
    .reply = null
}
- add the new_response to the lastResponseNode. 
lastResponseNode.response = new_response
</Add Response to Issue>

<Add Response to Response>
- create a Message message = {
    .author = agent | user
    .body = <the text>
    .timestamp = now()
    .seen = null
}
- Create a Response new_response = {
    .content = message,
    .responding_to = the Response being responded to
    .response = null
    .reply = null
}
- append to the end of the response chain:
    lastNode = theResponseBeingRespondedTo
    while( lastNode.response != null )
        lastNode = lastNode.response
    lastNode.response = new_response
</Add Response to Response>

<Add Reply to Response>
- create a Message message = {
    .author = agent | user
    .body = <the text>
    .timestamp = now()
    .seen = null
}
- create a Response new_response = {
    .content = message,
    .responding_to = the Response being replied to
    .response = null
    .reply = null
}
- if theResponseBeingRepliedTo.reply == null:
    theResponseBeingRepliedTo.reply = new_response
  else:
    // reply chain already exists — append to its tail
    lastNode = theResponseBeingRepliedTo.reply
    while( lastNode.response != null )
        lastNode = lastNode.response
    lastNode.response = new_response
</Add Reply to Response>
